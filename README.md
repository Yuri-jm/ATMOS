# ATMOS – Sistema de Previsão do Tempo

Aplicação web responsiva para consulta de previsão do tempo em tempo real, desenvolvida em React, com autenticação de usuários e integração com a API OpenWeather.

O sistema permite que usuários autenticados pesquisem cidades, visualizem informações climáticas atualizadas e gerenciem cidades favoritas, com foco em usabilidade tanto em desktop quanto em dispositivos móveis.

##  Objetivo do Projeto

Desenvolver uma aplicação web moderna e responsiva que consuma dados de uma API pública, implemente autenticação real de usuários e ofereça uma experiência fluida em diferentes dispositivos, aplicando conceitos de React, Scrum e boas práticas de front-end.

---

##  Tecnologias Utilizadas

- **React + Vite**
- **Firebase Authentication**
- **Firebase Hosting**
- **OpenWeather API**
- **Vitest + React Testing Library**
- **HTML5 / CSS3 / JavaScript**
- **Git & GitHub**

---

##  Funcionalidades

- Autenticação de usuários (login com e-mail e senha)
- Navegação entre telas (Login / Clima)
- Busca de cidades com consumo de API externa
- Exibição do clima atual:
  - Temperatura
  - Sensação térmica
  - Umidade
  - Velocidade do vento
- Gerenciamento de cidades favoritas
- Interface responsiva (desktop e mobile)
- Testes automatizados de componentes
- Controle de acesso a rotas com base na autenticação
